## Usage Example

```python
from text_preprocessing import TextPreprocessor

# Initialize
prep = TextPreprocessor()

# Load text
text = prep.read_text('text.txt')

# Full pipeline
clean_text = prep.preprocess_pipeline(text)

# Tokenize
tokens = prep.tokenize_words(clean_text)

# Remove stopwords
filtered = prep.remove_stopwords(tokens)

# Analyze
freq = prep.get_word_frequency(filtered)
print(freq.most_common(5))
```
